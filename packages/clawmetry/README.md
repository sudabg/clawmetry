# clawmetry

`clawmetry` is the brand alias package for `openclaw-dashboard`.

Install:

```bash
pip install clawmetry
```

Run:

```bash
clawmetry
```

This package depends on `openclaw-dashboard` and exposes the same dashboard runtime.
